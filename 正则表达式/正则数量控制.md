数量控制的编写方式
---
| 逻辑符          | 描述                     |
| --------------- | ------------------------ |
| `{n}`指定字符   | 匹配 n 个字符            |
| `{n,m}`指定范围 | 匹配[n,M]个字符          |
| `{n,}`n个以上   | 匹配n个或者n个以上的字符 |
| `*`任意个字符   | 相当于`{0,}`零或者无数个 |
| `+`至少有一个   | 相当于`{1,}`             |
| `?`0或者1个     | 相当于`{0,1}`            |

数量控制和[[表达式字符]][[正则逻辑控制]]结合的方式多种多样同时可以搭配[[正则逻辑控制#子表达式]]进行使用
